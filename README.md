# winQRcode_ch

我为什么要开发一个这样的软件呢？
因为这个软件好简单啊肯定一下子就写出来了于是我就开发了！

## 优点

* 占用内存小。速度相较网页更快。
  ![快速的连续输入](https://s2.ax1x.com/2020/01/20/1iCxM9.md.gif)
* 可以一定程度调节清晰度。
* 放大查看界面您生成的二维码仅需点击图片。
* 查看历史记录
* 完全绿色（因为我不会写病毒！）
* 随意保存

## 缺点

* 没有很好看的界面
* 不能换二维码的颜色
